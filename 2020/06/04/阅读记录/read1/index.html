B站-普通人如何实现财务自由

一、法则：

威廉班根 4%法则

二、投资方向参考：投资股票指数（需探讨）

 待补充

三、方法

  每年从投资中提取不超过4.2%的金额，用来支付生活所需

四、参考书籍：

《不上班也有钱》 — 作者设定40岁退休

五、思考模型及其思路：

结果为导向

  1. step1:
    制定目标，假设设计为，每年花费大概15W购买力附近的支出（这里的15万是2020年基准）
  2. step2:
    用15万除以4%得出本金375万。银行4%收益的产品不少，股票、指数基金相银行而言更加灵活，可以做组合投资，可能会说股票有涨有跌，这类投资属于长期投资，一般要选择比较标杆、大平台的投资产品，小周期性的亏损，在大周期的下算不了什么，不能因为说今年股票跌的惨，一辈子也不买股票这种说法，因为当选择这类投资时，一般是以3-5年为一个结算周期的。并且，想实现财务自由，投资也不可能一夜暴富，这背后都是满满的计算呀。
  3. step3:
    如何到达375万，单纯靠出卖劳动力来累积这375万是不可行的，对于年轻人而言，首要是存钱。
    存钱的逻辑公式是： 增加收入 <- 工作拼搏 <- 开源节流 -> 转为资产 -> 获得复利
    其次是发展副业，副业的逻辑公式是：增加收入 -> 加大投资 -> 增加复利 -> 资产增长 -> 财务自由
    总结：在年轻有资本的时候，把时间换为资产，是资产，不是工资，不是钱，也不是负债。
    高收入，高投资，后期追求高回报，实现财富自由（以上忽略了生儿育女的消耗）
  • 需要探讨的地方:
    1、自己每一年、每一个月、每一天的支出
    2、市面上，指数基金、理财等产品的一个收益情况，作出均值（数据说话）
    3、考虑一些其他的因素，例如：生孩子花费，结婚花费，买租房，买车等消耗。
    4、结合自身状况，实际情况去考虑这个模型哪里需要做出一些让步或者修改。
